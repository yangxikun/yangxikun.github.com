递归与分治--合并排序

01 October 2013

思想

将待排序元素分成大小大致相同的两个子集合,分别对两个子集合进行排序,最终将排好序的子集合合并成所要求的排好序的集合.

递归的归并排序

1 void MergeSort(Type a[], int left, int right){
2     if (left < right ) {//至少要有2个元素
3         int i = (left+right)/2;
4         MergeSort(a, left, i);//左半部分进行归并排序
5         MergeSort(a, i+1, right);//右半部分进行归并排序
6         Merge(a, b, left, i, right);//将a[left...i]和a[i+1...right]合并到辅助数组b中
7         Copy(a, b, left, right);//因为归并排序不是原址排序,所以要复制回数组a
8     }
9 }

归并排序非递归实现

需要设置一个归并的步长,起始为1,到n/2结束

 1 void MergeSort(Type a[], int n){
 2     Type *b = new Type[n];
 3     int s = 1;//增长步长,每次都以自身两倍增长
 4     while(s < n){
 5         MergePass(a, b, s, n);//归并到数组b中
 6         s += s;
 7         MergePass(b, a, s, n);//归并到数组a中
 8         s += s;
 9     }
10 }
11 
12 void MergePass(Type x[], Type y[], int s, int n){
13     //对数组x合并n/s个大小为s的相邻子数组到数组y中
14     int i = 0;
15     while (i < n) {
16         Merge(x, y, i, i+s-1, i+2*s-1);//合并大小为s的相邻两段子数组
17         i = i+2*s;
18     }
19     //剩下的元素个数少于2s
20     if (i+s < n) Merge(x , y, i, i+s-1, n-1);
21     else for (int j = i; j < n; j++) y[j] = x[j];
22 }
23 
24 void Merge(Type c[], Type d[], int l, int m, int r){
25     //合并c[l:m]和c[m+1:r]到d[l:r]
26     int i = l, j = m+1, k = l;
27     while (i <= m && j <= r) {
28         if (c[i] <= c[j]) d[k++] = c[i++];
29         else d[k++] = c[j++];
30     }
31     if (i > m) while (j <= r) d[k++] = c[j++];
32     else while(i <= m) d[k++] = c[i++];
33 }
