迭代模式

02 October 2013

学习自《Guide to PHP Design Patterns》

在PHP的SPL中,有iterator这个类库,它的主要用途在于为一组数据集(可以是数组或对象)提供各种遍历的方法.

一个简单的迭代

 1 <?php 
 2 class testIterator implements iterator {
 3     private $_store;//存储数据的集合
 4 
 5     public function __construct (&$arr) {
 6         $this->_store = $arr;
 7     }
 8 
 9     public function current () {
10         return current($this->_store);
11     }
12 
13     public function next () {
14         next($this->_store);
15     }
16 
17     public function key() {
18         return key($this->_store);
19     }
20 
21     public function rewind () {
22         reset($this->_store);
23     }
24 
25     public function valid () {//在next()和rewind()调用之后会被自动调用
26         return true;
27     }
28 }
29 $arr         = array(1,2,3,false,4,5);
30 $arrIterator = new testIterator($arr);
31 foreach ($arr as $key => $value) {
32     echo $key.'-'.$value.'<br />';
33 }
34 ?>

迭代的类库有各种迭代的接口,这也是为了规范开发者编写的代码吧,具体请看PHP手册iterator.
