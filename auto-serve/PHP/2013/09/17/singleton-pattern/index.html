单例模式

17 September 2013

学习自《Guide to PHP Design Patterns》

什么是单例模式?

一个类在整个PHP程序运行过程中,只需要被实例化一次,这些类通常是提供一些功能,且在程序中需要多次调用,最经典的莫过于数据库连接类.

单例模式示例

 1 <?php
 2 /**
 3  * 数据库连接类
 4  */
 5 class DbConn {
 6     /**
 7      * @static class 当前类的实例
 8      */
 9     static private $_instance = NULL;
10 
11     /**
12      * 私有构造器,使其无法通过new实例化类
13      * 
14      * @return NULL
15      */
16     private function __construct(){
17     }
18 
19     /**
20      * 返回当前类的实例
21      * 
22      * @static
23      * @return class 当前类的实例
24      */
25     static public function getInstance(){
26       if ( !self::$_instance ) {
27         DbConn::$_instance = new DbConn;
28       }
29       return self::$_instance;
30     }
31 }
32 ?>
