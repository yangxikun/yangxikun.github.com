多个文件上传

25 September 2013

html代码如下,有两点需要注意的:一是设置form的enctype属性,二是使用post方法

 1 <html>
 2 <head>
 3     <meta charset="utf-8">
 4 </head>
 5 <body>
 6     <form action="test.php" method="post" enctype="multipart/form-data">
 7         <input type="file" name="userfile1">
 8         <input type="file" name="userfile2">
 9         <input type="file" name="userfile3">
10         <input type="submit">
11     </form>
12 </body>
13 </html>

php代码如下:

 1 <?php
 2 /**
 3  * 错误代码:
 4  * 
 5  * UPLOAD_ERR_OK
 6  *
 7  *    其值为 0，没有错误发生，文件上传成功。
 8  * UPLOAD_ERR_INI_SIZE
 9  *
10  *    其值为 1，上传的文件超过了 php.ini 中 upload_max_filesize 选项限制的值。
11  * UPLOAD_ERR_FORM_SIZE
12  *
13  *    其值为 2，上传文件的大小超过了 HTML 表单中 MAX_FILE_SIZE 选项指定的值。
14  * UPLOAD_ERR_PARTIAL
15  *
16  *    其值为 3，文件只有部分被上传。
17  * UPLOAD_ERR_NO_FILE
18  *
19  *    其值为 4，没有文件被上传。
20  * UPLOAD_ERR_NO_TMP_DIR
21  *
22  *    其值为 6，找不到临时文件夹。PHP 4.3.10 和 PHP 5.0.3 引进。
23  * UPLOAD_ERR_CANT_WRITE
24  *
25  *    其值为 7，文件写入失败。PHP 5.1.0 引进。
26  *
27  */
28 if ( !empty( $_FILES ) ) {
29     foreach ( $_FILES as $key => $value ) {
30         if ( $value['error'] === 0 ) {//上传过程中未出错
31             if ( is_uploaded_file( $value['tmp_name'] ) ) {
32                 if ( move_uploaded_file( $value['tmp_name'], /*...目的位置*/ ) ) {
33                     echo 'file '.$value['name'].' has been uploaded.<br />';
34                 } else {
35                     echo 'Cann\'t upload file!Maybe permission deny!';
36                 }
37             }
38         }
39     }
40 }
41 ?>
