观察者模式

02 October 2013

学习自《PHP 精粹》

对于javascript开发者来说,这是一个再熟悉不过的模式了,因为js通过事件驱动来执行,例如页面加载,单击,鼠标移动等.

观察者模式的核心在于允许你的应用程序注册一个回调,当某个特定的事件发生时便会触发它.

下面例子通过一个Event类实现观察者模式:

 1 <?php
 2 /**
 3  * Event 类
 4  * 
 5  * 使用Event类,你可以为某个事件注册回调函数
 6  */
 7 class Event {
 8 
 9     /**
10      * @static
11      * @var array 键值对数组,键作为事件,值是存储事件的回调数组
12      */
13     static protected $callbacks = array();
14 
15     /**
16      * 注册一个回调
17      * 
18      * @param string $eventName 事件的名字
19      * @param mixed $callback 回调
20      * @return NULL
21      * @throws  Exception
22      */
23     static public function registerCallback($eventName, $callback) {
24         if (!is_callable($callback)) {
25             throw new Exception('Invalid callback');
26         }
27         $eventName = strtolower($eventName);
28         self::$callbacks[$eventName][] = $callback;
29     }
30 
31     /**
32      * 触发一个事件
33      * 
34      * @param string $eventName 事件名称
35      * @param mixed $data 传递给回调函数的数据
36      * @return NULL
37      * @throws  Exception
38      */
39     static public function trigger($eventName, $data) {
40         $eventName = strtolower($eventName);
41         if (isset(self::$callbacks[$eventName])) {
42             foreach (self::$callbacks[$eventName] as $key => $callback) {
43                 $callback($data);
44             }
45         } else {
46             throw new Exception('Invalid event');
47         }
48     }
49 }
50 
51 /**
52  * log回调
53  */
54 class LogCallback {
55     public function __invoke($data) {
56         echo 'Log data<br />';
57         var_dump($data);
58     }
59 }
60 
61 //注册回调
62 Event::registerCallback('save', new LogCallback());
63 //注册一个闭包
64 Event::registerCallback('save', function ($data) {
65                                     echo 'Clear Cache<br />';
66                                     var_dump($data);
67                                 });
68 class DataRecord {
69     public function save() {//触发事件
70         Event::trigger('save', array('Hello', 'world'));
71     }
72 }
73 
74 $data = new DataRecord();
75 $data->save();
76 ?>
