AOE网(关键路径)

07 September 2013

描述

AOE网是一个带权的有向无环图,其中,顶点表示事件,弧表示活动,权表示活动持续的时间.通常,AOE网可用来估算工程的完成时间.

网中只有一个入度为零的顶点(源点)和一个出度为零的顶点(汇点).

问题

1.完成整项工程至少需要多少时间? 2.哪些活动是影响工程进度的关键?

思想

  1. AOE网中有些活动是可以并行地运行,所以完成工程的最短时间是从开始点到完成点的最长路径的长度. 2.路径长度最长的路径叫做关键路径.
  2. 假设开始点是v1,从v1到vi的最长路径叫做事件vi的最早发生时间(因为vi事件要等到所有以vi为头的弧(活动)完成才能发生).这个时间决定了所有以vi为尾的弧所表示的活动的最早发生时间.
  3. 用e(i)表示活动ai的最早开始时间,用l(i)表示活动的最迟开始时间(在不推迟整个工程的前提下,活动ai的最迟开始时间),两者之差l(i)-e(i)意味着完成活动ai的时间剩余量.我们把l(i)=e(i)的活动叫做关键活动. 5.关键路径上的所有活动都是关键活动,因此提前完成非关键活动并不能加快工程进度.

求解分析

辨别关键活动就是要找e(i)=l(i)的活动.设AOE网中活动的最早开始时间为e(i)和最迟开始时间为l(i),事件的最早发生时间为ve(j)和最迟发生时间为vl(j).如果活动ai由弧j,k表示,其持续时间记为dut(j,k),则有如下关系:

e(i)=ve(j)

l(i)=vl(k)-dut(j,k)

求解ve(j)和vl(j)如下:

  1. 从ve(0)=0开始向前递推:ve(j)=Max{ve(i)+dut(i,j)} i,j属于T,T是所有以第j个顶点为头的弧的集合
  1. 从vl(n-1)=ve(n-1)起向后递推:vl(i)=Min{vl(j)-dut(i,j)} i,j属于S,S是所有以第i个顶点为尾的弧的集合

算法流程

  1. 输入e条弧,建立AOE网为邻接表存储结构
  2. 从源点出发,令ve0=0,按拓扑有序求其余各顶点的最早发生时间
  3. 从汇点出发,令vln-1=ven-1,按逆拓扑有序求其余顶点的最迟开始时间
  4. 根据各顶点的ve和vl,求每条弧s的最早开始时间e(s)和最迟开始时间l(s),若某条弧满足e(s)=l(s),则为关键活动

辅助

栈T用于存储拓扑序列 ve数组记录事件最早开始时间,vl数组记录事件最迟开始时间

代码

 1 Status TopologicalSort(ALGraph G, Stack &T){
 2     //有向图G采用邻接表存储结构
 3     //若G无回路,则输出G的顶点的一个拓扑序列并返回OK,否则ERROR.
 4     FindInDegree( G, indegree );    //对各顶点求入度indegree[0...vernum-1]
 5     InitStack( S );
 6     InitStack( T );
 7     ve[0..G.vexnum-1] = 0;
 8     for( i=0; i<G.vexnum; ++i ) //建零入度顶点栈S
 9         if( !indegree[i] ) StackPush( S, i ); //入度为0者进栈
10     /*上面步骤可以在FindInDegree()函数中完成*/
11     count = 0 ; //对输出顶点计数
12     while( !StackEmpty( S ) ){
13         StackPop( S, i );StackPush( T, i );++count;   //i号顶点入栈T并计数
14         for( p = G.vertices[i].firstarc; p; p = p->nextarc ){//对i号顶点的每个邻接点的入度减1
15             if( !( --indegree[p->adjvex] ) ) StackPush( S, p->adjvex );//若入度为零,则入栈
16             if( ve[i]+*(p->info) > ve[p->adjvex] ) ve[k] = ve[i]+*(p->info);//相当于ve(j)=Max{ve(i)+dut(<i,j>)}
17         }//for
18     }//while
19     if( count < G.vexnum ) return ERROR;    //该有向图有回路
20     else return OK;
21 }//TopologicalSort

23 Status CriticalPath(ALGraph G){
24     //G为有向网,输出G的各项关键活动
25     if( !TopologicalOrder(G, T) ) return ERROR;
26     vl[0..G.vexnum-1] = ve[0..G.vexnum-1];//初始化顶点事件的最迟发生时间
27     while( !StackEmpty(T) ) //按逆拓扑序列求各顶点的vl值
28         for( StackPop( T, j ), p = G.vertices[j].firstarc; p; p = p->nextarc ){
29             k = p->adjvex; dut = *(p->info);
30             if( vl[k]-dut < vl[j] ) vl[j] = vl[k]-dut;
31         }//for
32     for( j=0; j<G.vexnum; ++j )//求每条边的最早开始时间和最迟开始时间
33         for( p = G.vertices[j].firstarc; p; p = p->nextarc ){
34             k = p->adjvex; dut = *(p->info);
35             ee = ve[j]; el = vl[k]-dut;
36             tag = (ee == el) ? 1 : 0;
37             printf( j, k, dut, ee, el, tag);//输出关键活动
38         }//for p
39 }//CriticalPath
